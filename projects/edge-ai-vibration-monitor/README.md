# Edge AI Vibration Monitor

A portfolio project demonstrating on-device machine-learning for fault detection
via vibration analysis. Complements the Smart Marine Buoy project by carrying the
PCB-level design, high-frequency sensing, and calibration work.

_Project structure and documentation in progress._
